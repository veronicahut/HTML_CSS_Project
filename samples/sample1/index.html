Mobile Application Assignment

This assignment focuses on the fundamentals of HTML and CSS for mobile app development. It includes a simple button that, when clicked, will trigger a JavaScript function to display an alert message. The purpose of this assignment is to demonstrate the ability to create a responsive app with basic interactivity using HTML, CSS, and JavaScript.
